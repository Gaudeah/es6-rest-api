# es6-rest-api
Simple and fresh ES6 REST API Written in JS, using new Async/Await protocol

## Configuration
Every configuration you need is located on ./config.js (Host, Port, Motd..)


Sample config.js
```json
module.exports = {
    "service": {
        "host": "http://localhost",
        "port": 3000,
        "motd": "Server listening at "
    },
}
```
